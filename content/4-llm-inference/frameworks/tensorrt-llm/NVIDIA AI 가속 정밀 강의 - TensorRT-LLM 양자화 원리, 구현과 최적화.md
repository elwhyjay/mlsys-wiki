# NVIDIA AI 가속 정밀 강의 - TensorRT-LLM 양자화 원리, 구현과 최적화

NVIDIA AI 가속 정밀 강의 - TensorRT-LLM 양자화 원리, 구현과 최적화

> TensorRT-LLM은 풍부한 대규모 언어 모델(LLM)을 포괄하고 업계의 선진 가속 알고리즘을 지원할 뿐만 아니라, 효율적인 엔지니어링 최적화까지 구현해 고성능 LLM을 진정으로 "바로 사용할 수 있게" 만든다. 이번 공유는 "개봉 리뷰"로서 "Hello World"부터 시작해 서너 줄의 명령으로 자신의 대규모 언어 모델을 실행하고, TensorRT-LLM 사용 흐름을 체험하며, LLM 성능 튜닝의 best practice를 소개한다.
